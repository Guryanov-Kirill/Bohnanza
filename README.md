# Bohnanza

Приложение для администрирования карточной игры Bohnanza на Kotlin (Gradle).

---

## Правила игры

Bohnanza — карточная игра на торговлю и выращивание бобов для 2–7 игроков.

**Компоненты:** 104 карты бобов 8 видов (чем реже вид — тем ценнее), монеты на обороте карт.

**Цель:** набрать больше всего монет, продавая урожай бобов.

**Поля игрока:** у каждого 2 поля для выращивания. Каждый игрок может купить третье поле за монеты. На каждом поле может быть только один вид бобов.

**Ключевое правило:** карты в руке нельзя переставлять. Брать можно только с конца, сажать только с начала.

---

## Ход партии

Каждый ход состоит из четырёх шагов:

1. **Посадка (обязательно)** — сыграть первую карту из руки на одно из полей (можно и вторую, но не больше).

2. **Тянуть 2 карты** — открыть 2 карты из колоды лицом вверх.

3. **Торговля** — активный игрок торгует открытыми картами с другими игроками. Можно отдавать и получать любые карты из руки. Оба участника сделки обязаны немедленно посадить полученные карты.

4. **Добор** — взять 3 карты из колоды в руку (в конец руки — порядок менять запрещено).

**Продажа урожая (Harvest):** чтобы освободить поле или посадить несовместимый боб, нужно продать всё поле. Количество монет определяется по таблице на каждой карте. Невыгодная продажа (1 боб = 0 монет) просто выбрасывается.

**Конец игры:** когда колода заканчивается в третий раз — все игроки продают поля и считают монеты. Побеждает тот, у кого их больше.

---

### Игровой цикл

```
Game.start()
|
+-- loop: Game.nextTurn()
    |
    |-- PLANT_MANDATORY - обязательная посадка первой карты из руки****
    |
    |-- DRAW_AND_TRADE - открыть 2 карты, торговля с игроками
    |
    +-- DRAW_FROM_DECK - добрать 3 карты в руку
        |
        +-- Deck.isThirdCycle() == true -> Game.finish()
```
